# 🛒 Tation清单 (Tation List)

Tation清单是一款轻量、优雅的移动端清单工具。专为聚会、露营、食材采购等场景设计，深度适配 iPhone Pro Max 等主流机型。

## ✨ 特色功能
- **全本地存储**：数据存在你的手机里，隐私且无需登录。
- **长图分享**：一键生成纯净清单图片，朋友圈或微信分享更美观。
- **无损同步**：通过“导出/导入”文本，轻松实现多设备数据迁移或好友间清单共享。
- **iOS 优化**：针对 iPhone 灵动岛及底部安全区域进行了完美适配。

## 🚀 进阶玩法：如何同步给好友？
1. **你**：点击右上角分享图标 -> 点击“导出清单”，一段清单代码会自动复制到你的剪贴板。
2. **发送**：将这段代码通过微信发给好友。
3. **好友**：在微信里长按“全选并复制”那段代码，打开本网页，点击分享图标 -> “导入清单”即可。

## 💡 使用小贴士
- **添加到主屏幕**：在 Safari 浏览器中点击底部的“分享”按钮 -> “添加到主屏幕”，即可像原生 App 一样使用。
- **数据安全**：本工具不联网。如果更换手机或清理浏览器深度缓存，数据会消失。建议重要清单定期使用“导出”功能保存一段文本作为备份。

## 🛠️ 技术栈
- Vue 3 (Composition API)
- Vant 4 UI Library
- html2canvas (长图生成)
