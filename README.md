# Tation清单 - 移动端食材管理工具

Tation清单是一款专为手机设计的购物/准备清单工具。数据完全存储在本地浏览器，无需登录，隐私安全。

## 🌟 核心功能
- **模块化管理**：支持创建多个大类（如：火锅、烧烤）。
- **智能分类**：内置肉类、蔬菜、底料等常用分类快捷选择。
- **价格无关设计**：专注于清单勾选，适配 iPhone 15 Pro Max 等大屏布局。
- **长图导出**：一键生成高清清单图片，方便发给微信好友。
- **数据同步**：支持通过“加密代码”导出和导入数据，实现手动多端同步。

## 📱 手机端最佳使用方式
1. 用 Safari 或 Chrome 打开你的 GitHub Pages 链接。
2. 点击浏览器下方的 **“分享”** 按钮。
3. 选择 **“添加到主屏幕”**。
4. 现在你可以像使用 App 一样随时从桌面打开它。

## 💾 数据说明
- 本工具不使用数据库，数据保存在手机浏览器的 `LocalStorage` 中。
- **注意**：清理浏览器缓存可能会导致数据丢失，请定期使用“导出数据”功能备份重要清单。
